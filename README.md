# üìù Perception - AI-Powered Online Evaluation Portal

**Perception** is a full-stack web application designed to revolutionize the educational assessment process. It leverages a powerful Large Language Model (LLM) to provide instant, accurate, and personalized feedback for descriptive answers, bridging the gap between automated grading and human-like evaluation.

## ‚ú® Features

### For Students
- **Role-Based Authentication:** Secure login and signup, including Google Authentication.
- **Personalized Dashboard:** View available assignments from all teachers and track submitted work.
- **Dynamic Assignments:** See assignments that are public or specifically assigned to you by a teacher.
- **Real-Time AI Feedback:** Submit answers and receive an instant score and constructive feedback generated by an LLM.
- **Final Grades:** View finalized scores after a teacher's review.

### For Teachers
- **Secure Authentication:** Login and signup system to manage course materials.
- **Question Set Management:** Create, view, and manage descriptive question sets with model answers to guide the AI.
- **Student Assignment:** Assign question sets to all students (publicly) or to specific students by their username.
- **AI-Assisted Grading:** View a comprehensive list of student submissions with AI-generated scores and feedback.
- **Review & Finalize:** Override the AI's suggestion and finalize the score for any submission.

## ‚öôÔ∏è Getting Started

To get a local copy up and running, follow these simple steps.

### Prerequisites

- Python 3.10+
- Node.js 20.x
- A MongoDB Atlas account
- A Groq API Key
- A Google Cloud Platform account for OAuth credentials

### Backend Setup (`perception-api`)

1.  **Clone the repository:**
    ```sh
    git clone [https://github.com/your-username/perception.git](https://github.com/your-username/perception.git)
    cd perception/perception-api
    ```
2.  **Create and activate a virtual environment:**
    ```sh
    python -m venv venv
    source venv/bin/activate  # On Windows: venv\Scripts\activate
    ```
3.  **Install dependencies:**
    ```sh
    pip install -r requirements.txt
    ```
4.  **Create a `.env` file** in the `perception-api` root and add your environment variables:
    ```env
    DATABASE_URL="your_mongodb_atlas_connection_string"
    SECRET_KEY="a_very_strong_random_secret_key"
    ALGORITHM="HS256"
    ACCESS_TOKEN_EXPIRE_MINUTES=30
    GROQ_API_KEY="your_groq_api_key"
    GOOGLE_CLIENT_ID="your_google_client_id"
    ```
5.  **Run the server:**
    ```sh
    uvicorn app.main:app --reload
    ```

### Frontend Setup (`perception-ui`)

1.  **Navigate to the frontend directory:**
    ```sh
    cd ../perception-ui
    ```
2.  **Install dependencies:**
    ```sh
    npm install
    ```
3.  **Create a `.env.local` file** in the `perception-ui` root and add your environment variables:
    ```env
    NEXT_PUBLIC_API_URL="[http://127.0.0.1:8000](http://127.0.0.1:8000)"
    NEXT_PUBLIC_GOOGLE_CLIENT_ID="your_google_client_id"
    ```
4.  **Run the development server:**
    ```sh
    npm run dev
    ```
